Complete User Guide

Welcome to Arch, your comprehensive productivity companion! This guide will help you master all the features of this beautiful and powerful todo app.

📱 Getting Started

First Launch

  1. Splash Screen: The app starts with a beautiful animated splash screen featuring the Arch logo
  2. Authentication: You'll need to create an account or sign in to use the app
  3. Welcome: After login, you'll receive a welcome notification

Authentication Options

  • Email/Password: Create a new account or sign in with existing credentials
  • Google Sign-In: Quick and secure login with your Google account
  • Registration: New users can create an account with email and password

🏠 Main Interface

Navigation Bar

The app features a modern bottom navigation with four main sections:

  1. 📋 Tasks - Your main task management hub
  2. 📅 Calendar - View tasks by date
  3. 📊 Stats - Productivity analytics
  4. ⏰ Timer - Pomodoro and custom timers

App Bar Toggle

  • Tap the small notch at the top center to expand/collapse the app bar
  • The app bar shows your profile, greeting message, and quick access buttons

📋 Task Management

Creating Tasks

  1. Tap the + button in the Tasks tab
  2. Fill in task details:
    • Title (required): Enter your task name
    • Description (optional): Add detailed notes
    • Due Date: Select when the task is due
    • Important: Toggle to mark as priority task
  3. Tap "Add Task" to save

Task Features

  • Priority Marking: Important tasks show a red priority icon
  • Categories: Tasks can be organized by categories (color-coded)
  • Due Dates: Each task has a specific due date
  • Completion Tracking: Check off completed tasks

Managing Tasks

  • Complete: Tap the checkbox to mark as done
  • Edit: Long press any task to edit details
  • Delete: Swipe left on a task and tap the delete button
  • View Details: Tap on a task to see full information

📅 Calendar View

Calendar Features

  • Monthly View: See all your tasks on a calendar
  • Daily Overview: Tap any date to see tasks for that day
  • Task Markers: Days with tasks show colored dots
  • Format Options: Switch between month, 2-week, and week views

Using the Calendar

  1. Navigate: Swipe or use arrows to change months
  2. Select Date: Tap any date to view its tasks
  3. View Tasks: See all tasks scheduled for the selected date
  4. Quick Actions: Mark tasks complete directly from calendar view

📊 Statistics & Analytics

Productivity Overview

  • Total Tasks: Count of all created tasks
  • Completed Tasks: Number of finished tasks
  • Pending Tasks: Remaining incomplete tasks
  • Productivity Percentage: Overall completion rate

Performance Record

  • Best Streak: Longest consecutive days with 100% completion
  • Weekly Rate: Current week's completion percentage
  • Daily Progress: Visual chart of last 7 days' performance
  • Color Coding: Green (100%), Orange (50%+), Red (<50%)

⏰ Timer Features

Pomodoro Timer

  • Focus Sessions: 25-minute work periods
  • Break Sessions: 5-minute rest periods
  • Automatic Switching: Seamlessly alternate between work and break

Using Pomodoro Timer

  1. Start Focus: Tap play to begin a 25-minute work session
  2. Take Breaks: Timer automatically switches to 5-minute breaks
  3. Manual Control: Use buttons to switch between work/break modes
  4. Reset: Tap refresh to reset the timer

Custom Timer

  • Flexible Duration: Set any time from 1 minute to 2 hours
  • Simple Controls: Use +/- buttons or type directly
  • Personalized Sessions: Perfect for custom work intervals

👤 Profile & Settings

Profile Management

  • Edit Name: Tap the edit icon next to your name
  • Change Avatar: Choose from anime avatars or upload from gallery
  • View Email: Your registered email address
  • Logout: Secure sign-out option

Avatar Options

  • Anime Avatars: 7 pre-designed anime-style avatars
  • Gallery Upload: Use your own photos
  • Quick Selection: Easy avatar switching

🎨 Theme & Customization

Light Mode

  • Clean Interface: Bright, modern design
  • Animated Backgrounds: Beautiful gradient transitions
  • Glassmorphism Effects: Translucent UI elements
  • High Contrast: Easy readability

Dark Mode

  • Eye-Friendly: Reduced eye strain in low light
  • Battery Efficient: Saves battery on OLED screens
  • Modern Aesthetic: Sleek dark interface
  • Consistent Design: Same features, different appearance

🚀 Tips & Best Practices

Productivity Tips

  1. Use Categories: Organize tasks by project or type
  2. Set Realistic Due Dates: Don't overload single days
  3. Mark Important Tasks: Prioritize what matters most
  4. Use the Timer: Focus on one task at a time
  5. Check Statistics: Monitor your progress regularly

App Usage Tips

  1. Long Press: Use long press for editing tasks
  2. Swipe Gestures: Swipe left to delete tasks
  3. Calendar Navigation: Use calendar for date-based planning
  4. Theme Switching: Choose the theme that works best for you
  5. Regular Backups: Your data syncs automatically
